## Commands

Install node packages

```bash
npm install
```

Build the library

```bash
npm run build
```

Build the document for src code

```bash
npm run doc
```

Push src code to apps script project

```bash
npm run push
```

Pull src code from apps script project

```bash
npm run pull
```

Open apps script porject

```bash
npm run open
```
